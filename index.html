You may want to consider the following languages:

C#

C# can be used to create almost anything but is particularly strong at building Windows desktop applications and games. C# can also be used to develop web applications and has become increasingly popular for mobile development too. Cross-platform tools such as Xamarin allow apps written in C# to be used on almost any mobile device.

Java

Since it has been around for so long, some of the biggest organizations in the world are built using the language. Many banks, retailers, insurance companies, utilities, and manufacturers all use Java. Java is still a relevant programming language that shows no sign of declining in popularity and, for that reason, is worth learning. Many developers pick it up as their first programming language because itâ€™s reasonably easy to learn.

PHP

PHP is a programming language used for creating dynamic websites. PHP is known for its ability to optimize memory. As a result, it never allows server overloading. Instead, it helps in improving a server's processing speed significantly.

Ruby

Ruby has become popular because of the Ruby on Rails framework for web applications. With this framework, people who know Ruby can quickly create interactive web applications.

Choosing your first programming language isn't as important as you might think. Once you learn programming it will be much easier to pick up additional languages, and it may even be expected of you. Research what types of things you want to do, and find out what languages they use. Perhaps try out other languages while you are learning your first so you can be comfortable switching between them.
